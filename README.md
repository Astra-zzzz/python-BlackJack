

## BlackJack


#This is a terminal based BlackJack game based in Python as a beginner project


#Features:
-OOP Design
-Simple Dealer AI
-Ace Handling (1/11)
-Duplicate prevention
-Repeatable playthrough
-Win Counter


#Requirements:
-python 3.10 or higher
-install rich
    (installing rich:
    run:    pip install rich
    in bash/terminal)


#How to run:
run:
    python main.py
from the project folder, and in bash/terminal
